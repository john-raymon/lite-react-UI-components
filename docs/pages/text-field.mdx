# Text Field (with inline labels)
<br/>
The text field element by default renders as an input with `type="text"`. 
<TextField className="mt-2"/>
<br/>
Optionally a placeholder prop can be provided, while still rendering well with the inline label.
<TextField className="mt-2" placeholder="The label and placeholder can coexist"/>
<br/>
<br/>

## TextField as Text Area
<TextField className="mt-2" type="textarea" label="Label for text-area" placeholder="This is a placeholder for a text-area"/>
<br/>

### Basic example

```js
import { useState } from "react";
import { TextField } from "lite-react-ui";
import "./styles.css";
import "lite-react-ui/dist/style.css";

export default function App() {
  const [value, setValue] = useState("hello world");
  return (
    <div className="App">
      <TextField
        value={value}
        onChange={(e) => {
          setValue(e.target.value);
        }}
      />
    </div>
  );
}
```
